Welcome to the Spokane Mayor's Cyber Cup 2019!

We at Gonzaga University's Makers and Developers (GU MAD) club are super hyped that you are interested in computer security and contributing to a learning environment that will benefit us all. It gives us great pleasure to be hosting this event! We've prepared a variety of challenges, including an attack-and-defend competition and Hacker's Jeopardy.

To register, please go here and add your team. You may register with up to three members, or five if two of your team members are underclassmen.

Preparation

It would be wise to be familar with the following technologies when competing:
  • SSH: Teams will connect to administrate the servers using SSH and log into wargame challenges.
  • Linux: The attack/defend boxes will be running a Ubuntu 16.04 LTS. Teams should be familar with these basic principles:
  • Basic directory navigation
  • Command line text editor (vim, emacs, nano, etc.) Only takes a few minutes to pick one of these classic text editors up!
  • Passion to learn: The most important skill to have!

    • These skills would be nice...But, do not panic! This is suppossed to be a learning experience for everyone involved! You are not expected to know these going in.

  • Web Application Development:
  • NodeJS
  • Javascript
  • HTML
  • Python: Version 2.7
  • C/C++: Use the gcc and g++ compilers in Linux.
  • Assembly: x86 architecture knowlege of general registers, stack and how to use GDB.
  • Cryptography: RSA encryption and basic understanding of a symmetric key cipher.
  • Networking: Be able to read and understanding a pcap file
  • Social Engineering: How are people manipulated? What does a phishing email look like? Types of vulnerabilities that students should watch out for:
  • Directory traversal
  • SQL Injection
  • Buffer overflows
  • Race conditions
  • Access Control Issues
  • Poorly written logic
  • Uncontrolled format string
  • XML External Entity (XXE)
